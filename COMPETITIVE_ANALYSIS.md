# 競品分析報告 (Competitive Analysis)

針對目前市場上的健身輔助產品與開源專案，本專案之定位與差異化分析如下：

## 1. 競品分類對比

| 類別 | 代表產品 | 優勢 | 本專案差異化 (USP) |
| :--- | :--- | :--- | :--- |
| **智慧硬體** | Mirror, Tonal | 感測器極其精準，已有部分市場 | **低硬體成本**，僅需視訊鏡頭，且運動器材還是以徒手或健身房槓鈴啞鈴為主。 |
| **行動 App** | Kaia Health | 客製化訓練、真人評估 | **強調力量訓練細節** 著重於動作品質分析。 |
| **開源專案** | AI-Fitness-Trainer | 準確度高 | **具備 OOP 擴充性**在深蹲及伏地挺身的基礎下，希望加入臥推等動作。 |

## 2. 本專案核心價值 (Core Value)

* **動作解析**：不只是計次，透過向量運算 (NumPy) 分析關節角度，提供即時品質回饋。
* **開發結構**：採用 OOP 抽象類別設計，可擴充不同運動項目（如從伏地挺身擴充至臥推）。
* **記錄個人訓練成果**：長期記錄個人訓練組次數，量化訓練成效。

## 3. 結論
本專案旨在提供一個低門檻、具備量化數據的虛擬教練專案，補足目前市場上缺乏重量訓練、長期追蹤成效等缺口。
